Carta de presentacion personal


Objetivos Academicos sobre la materia:

  • Aprender a desarrollar aplicaciones usando Laravel
  • Desarrollar aplicaciones con validaciones de datos con Laravel
  • Estructurar un proyecto con Laravel que se pueda comunicar con otros lenguajes de programación.
      • Objetivos academicos en general:
    • Poder finalizar mis estudios de Ingenieria con exito
    • Obtener un CUM arriba de 9.5
    • Poder realizar cursos que le den mayor valor a mi curriculum
    • Desarrollar proyectos aplicando todo lo estudiado

Objetivos Personales:

  • Establecer una buena comunicación con los compañeros de clase
  • Cumplir con las asignaciones establecidas en las sesiones de clase.
  • Aprender en la medida de lo posible, el mayor numero de contenidos.
  • Comprender y aplicar los contenidos posibles en el entorno de aplicación.

Competencias adquiridas:

  • Creacion de proyectos en Laravel
  • Aprendizaje y comprension de Rutas y directivas dentro de un proyecto de Laravel
  • Creacion de un CRUD con Laravel
  • Aplicacion de JetStream y LiveWire en un proyecto de Laravel para la validacion de datos
  • Creacion de una API REST en LARAVEL para trabajar el Backend de los proyectos
  • Fusion de una API de Laravel como Backend y un proyecto de VueJs como Frontend

Contenidos mas dificiles y dificultades:

  • Creacion de la API REST con Laravel
  • Integración de API REST creada con Laravel en un proyecto de VueJS
  • Creacion de un CRUD con JetSteam y Livewire
